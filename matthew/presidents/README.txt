This program is intended to test knowledge of the presidents.

Compiling/Using the makefile:
  Compile with "make" and use "make clean" to delete the executable
  
Running:
  Run with "./presidents" or "./presidents.exe"

Flags:
  -reset - Resets the statistics file

Associated files and their purposes:
  presidents.cpp - The program itself
  config.ini - Edit to change settings
  presidents.txt - Edit or view stored information on the presidents
  stats.txt - View statistics of correct or incorrect answers - key is provided in the file
  stats_template.txt - Used to store the template for the statistics file
